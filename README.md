# PepperWallow

# Pre-requisites
- [Node 14](https://nodejs.org/en/download/) 

# Install
1. Clone the project
2. Navigate to the directory
3. Open it in a command line utility (I personally use powershell)
4. execute `npm run install` and answer the prompts.
5. Profit!
